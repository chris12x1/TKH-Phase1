# Week 3 — Security Automation with Python

## 🎯 Objective
Develop automation tools to detect, analyze, and respond to security threats using Python.

---

## 🛠️ What I Built

- Built a Python-based port scanning tool using sockets to identify open and closed services across multiple hosts
- Developed a log analysis script to detect brute-force authentication attempts and extract relevant entries
- Automated system auditing by executing and parsing system processes using subprocess
- Generated structured JSON alerts to standardize threat reporting for downstream analysis
- Created an end-to-end incident response script to extract attacker IPs from logs and produce machine-readable intelligence
  
---

## 📂 Artifacts

- `brute_detector.py` → Brute-force detection and log filtering script  
- `brute_report.txt` → Detection results
- `incident_response.py` → Automated threat detection and JSON reporting
- `port_check.py` → Network port scanning tool      
- `security_alert.json` → Alert output
- `system_auditor.py` → Process auditing and alert generation tool
- `threat_report.json` → Structured threat report
